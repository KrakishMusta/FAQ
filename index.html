О нас

Целью проекта является создание ресурса, за счёт которого будет достигнут высокий уровень осведомлённости населения о исторических объектах Кузбасса.

На данной странице представлены основные функциональные возможности приложения проекта.

Функционал приложения

Функционал главной страницы

На данной странице располагается карта с объектами.

На карте имеются метки объектов (метки объектов расположены условно). Метки располагают следующим функционалом: при наведении на них изменяют цвет на более светлый оттенок, при нажатии на метку. При нажатии на метку отображается оверлей с информацией об ассоциированном с ней объекте. Оверлей содержит кнопку для перехода на Странцу объекта и кнопку для отображения объекта на карте другого сервиса (Показать на карте). Локации, где метки расположены географически близко, так, что масштабы карты не позволяют графически верно и удобно для пользователя, могут сочетать в себе метки и имеют отличный функционал от других меток.

Функционал страницы объекта

Страница объекта располагает такими средствами интрерактивности с пользователем как:

  • 3D-модель, которая по умолчанию скрыта, становится видимой при нажатии кнопки 3D-модель.
  • Детальное описание основных черт объекта (год основания, особенности объекта).
  • Карта событий объекта. Данная карта иллюстрирует основные важные события в хронологическом порядке. Можно меремещаться меж событиями нажимая на условное событие слева от первоначального, чтобы перейти к предыдущему событию, на правое событие, перейти к следующему (по хронологическому порядку). Каждое событие состоит из года события и краткого его описания.
  • Кнопка для просмотра объекта на подробной карте другого сервиса.

Функционал пользовательских подсказок

Над основными функциональными элементами автоматически появляются подсказки (указатели), цель которых заключается в указании пользователю на возможность с ними взаимодействовать.

Подсказки можно будет отключить нажав по ним.

Пути развития приложения

Приложение имеет общирный потенциал развития. При поддержки соответствующих обществ и ведомств, можно организовать автоматизированную систему, способную предоставлять пользователям приложения полную информацию об объектах области.
